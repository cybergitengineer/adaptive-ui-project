# Adaptive UI Project

## Overview
This project demonstrates an adaptive user interface that changes based on user interaction. The system tracks which features the user selects most often and then prioritizes the most frequently used feature in the dashboard.

## Features
- Tracks user interactions using browser localStorage
- Identifies most-used feature
- Reorders interface dynamically
- Highlights preferred feature
- Displays usage statistics
- Allows reset of adaptation data

## Files
- `index.html` - main interface
- `style.css` - visual design
- `script.js` - adaptive logic
- `README.md` - project documentation

## How to Run
1. Download or clone the repository
2. Open `index.html` in a browser
3. Click different dashboard features
4. Observe how the interface adapts

## Adaptive Logic
The system uses interaction frequency as a simple AI-related adaptation mechanism. As the user selects features, the system learns preferences and changes the interface accordingly.
